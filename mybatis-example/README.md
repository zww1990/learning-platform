# Mybatis

* 如果是在eclipse中，选中pom.xml文件，点击右键选择 Run AS ——> Maven Build… ——> 在Goals框中输入：mybatis-generator:generate
* 如果在命令行输入Maven命令即可，注意：一定是当前项目目录下运行该命令：mvn mybatis-generator:generate
* 使用Java运行 MyBatis Generator
